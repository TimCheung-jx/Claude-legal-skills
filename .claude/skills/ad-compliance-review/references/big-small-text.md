# "大字吸睛、小字免责"风险提示

## 监管背景

2026年3月13日，市场监管总局印发《关于强化广告中提示性用语监管工作的通知》，部署对"大字吸睛、小字免责"等广告乱象开展为期半年的清理整治工作。

此前，2025年12月12日市监总局已发布《广告引证内容执法指南（征求意见稿）》，首次在官方文件中明确提及并规制该现象。

---

## 什么是"大字吸睛、小字免责"

指经营者在广告中以醒目、显著方式突出对其有利的信息（如功效、销量、折扣等），同时将**限制条件、重要前提、风险提示、例外情形**等关键信息以以下方式弱化呈现：

- 减小字号
- 改变字体
- 使用与背景相近颜色
- 位置隐蔽（边缘、角落）
- 展示时间极短（视频广告）
- 高度专业化表述（让消费者难以理解）

**本质**：信息披露方式造成实质性误导，不是单纯的"形式问题"。

---

## 典型违法形式

### 形式1：销量/排名类
```
大字：全网销量第一
小字：*指2023年某电商平台某类目单日销量
```
**风险**：🟡 高风险——引证信息范围限缩，可能构成虚假宣传

### 形式2：折扣优惠类
```
大字：全场1折起
小字：*仅限部分商品，详情请咨询店员
```
**风险**：🔴 明确违法——未显著标明提示性用语

### 形式3：功效限定类
```
大字：逆光之王
小字：*逆光之王是产品设计目标
```
**风险**：🟡 高风险——对产品性能作出限缩解释，误导消费者

### 形式4：数据宣称类
```
大字：2200MPa超强钢
小字：*项目名称
```
**风险**：🟡 高风险——技术指标实为内部代号，非真实性能

### 形式5：金融借贷类
```
大字：零利息
小字：*仅限新人，仅限前三个月
```
**风险**：🔴 明确违法——弱化对消费者不利信息

## 法律风险评级

| 行为 | 风险等级 | 法律依据 |
|------|----------|----------|
| 用极小字号/隐蔽位置标注限制条件 | 🔴 明确违法 | 《广告法》第8条、第11条 |
| 用与背景相近颜色隐藏重要信息 | 🔴 明确违法 | 《广告法》第4条（诚实信用） |
| 引证范围限缩，制造"第一"噱头 | 🟡 高风险 | 《广告引证内容执法指南》 |
| 对性能作出不符合常理的解释 | 🟡 高风险 | 《广告法》第28条（虚假广告） |
| 弱化对消费者不利信息 | 🔴 明确违法 | 《消费者权益保护法》第20条 |

## 合规要求

### 必须显著标明的内容
以下信息不得使用"大字小字"方式弱化：

1. **优惠条件限制**：适用范围、时间限制、数量限制
2. **引证范围**：统计数据来源、统计周期、统计口径
3. **性能限定**：技术指标的测试条件、适用范围
4. **风险提示**：产品使用风险、禁忌事项
5. **责任免除**：经营者免责条款需显著提示

### 显著标明的标准

✅ **合规做法**：
- 字号：不小于主要宣传文字字号的1/2
- 位置：与主要信息同区域展示
- 颜色：与背景有明显对比度
- 时长：视频广告中展示时间充足
- 字体：清晰易读的常规字体

❌ **违规做法**：
- 字号过小（如大字24px，小字8px）
- 位置隐蔽（页面底部、边缘角落）
- 颜色淡化（浅灰底+白色小字）
- 视频一闪而过（不足3秒）
- 专业术语堆砌（消费者难以理解）

---

## 审查要点

审查广告时需特别关注：

1. **是否存在大小字对比**：目测广告中字号差异是否过大
2. **小字位置是否隐蔽**：是否位于边缘、角落、底部
3. **颜色对比度**：小字是否与背景色相近难以辨认
4. **内容限缩**：小字是否对大字的宣称作出限缩解释
5. **不利信息弱化**：是否将限制条件、风险提示放在不显眼位置

## 改写示例

**原文（违规）**：
```
【大字】10户中国家庭，7户用公牛
【小字】*数据来源：某调研公司2022年抽样调查
```

**改写（合规）**：
```
【同字号】10户中国家庭，7户用公牛
【显著标注】数据来源：××调研公司，2022年对××城市1000户家庭抽样调查
```

**原文（违规）**：
```
【大字】全场5折
【底部小字】*部分商品除外，具体以门店为准
```

**改写（合规）**：
```
【同区域同字号】全场5折（指定商品参与，详见店内公示）
```

## 典型案例警示

### 案例1：公牛集团案（2026年4月）
**广告语**："10户中国家庭，7户用公牛"
**争议点**：引证数据范围和统计口径存在争议
**状态**：被称为"大字吸睛小字免责第一案"，正在司法审理中
**警示**：数据宣称必须有充分、可验证的证据链

### 案例2：某手机品牌"逆光之王"
**广告语**：大字"逆光之王"，小字"逆光之王是产品设计目标"
**问题**：对产品性能作出限缩解释，误导消费者认为已实现该性能
**处罚**：被市监部门责令整改并处罚

### 案例3：某汽车品牌"超强钢"
**广告语**：大字"2200MPa超强钢"，小字"项目名称"
**问题**：技术指标实为内部研发代号，非实际应用性能
**处罚**：涉嫌虚假宣传，立案调查
