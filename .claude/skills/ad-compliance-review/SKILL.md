# 广告合规审查

## 核心审查原则

### 规则分层（必须遵守）

| 层级 | 类型 | 示例 | 引用限制 |
|------|------|------|----------|
| **A层** | 硬规则 | 法律、行政法规、部门规章 | 可直接引用 |
| **B层** | 技术规则 | 强制性国标、推荐性国标 | 需标注性质 |
| **C层** | 监管口径 | 地方市监提示、典型案例 | 不得冒充A层 |
| **D层** | 实务经验 | 平台规则、风控建议 | 明确标注"建议" |

### 结论克制原则

| 风险等级 | 图标 | 定义 | 处理方式 |
|----------|------|------|----------|
| **明确违法** | 🔴 | 表达直接触法 | 必须删除或根本改写 |
| **高风险** | 🟡 | 极易被认定违法 | 建议删除或大幅修改 |
| **需证据支持** | 🟠 | 需证据链支撑 | 补充证明材料后可保留 |
| **表述不清** | 🔵 | 可能引发投诉 | 优化表述，消除歧义 |

## 标准审查流程（8步法）

### Step 1: 商业广告定性
判断四要素：营销性、媒介性、受众不特定性、非强制性

### Step 2: 行业识别与首轮信息采集
**重点行业**：儿童/母婴产品、教育培训、医疗器械、药品

**必须询问**：
1. 所属行业？
2. 广告形式（图文/视频/直播/海报/落地页）？
3. 发布地域？
4. 是否涉及数据宣称（第一/销量/专利）？
5. 是否存在"大字小字"排版？
6. 是否需要改写/Word报告？

### Step 3: 违禁词与高风险表述扫描
**绝对化用语（高风险）**：
- 国家级、最高级、最佳、第一品牌、遥遥领先
- 替代："优质"、"深受欢迎"、"行业领先之一"

### Step 4: "大字吸睛、小字免责"专项审查 ⚠️ 重点

**2026年3月新规**：市监总局部署半年专项整治

**典型违法形式**：
- 大字"全场1折"，小字"*部分商品参与"（底部灰色小字）
- 大字"10户家庭7户用XX"，小字"*来源：某调研公司抽样"
- 大字"零利息"，小字"*仅限新人，限前3个月"

**审查要点**：
1. 是否存在明显大小字号差异？
2. 限制条件是否使用极小字号/隐蔽位置？
3. 颜色是否淡化（如浅灰底+白色小字）？
4. 小字内容是否对大宣称作出限缩解释？

**风险评级**：
- 🔴 明确违法：用极小字号/隐蔽位置隐藏限制条件
- 🔴 明确违法：用与背景相近颜色弱化风险提示
- 🟡 高风险：引证范围限缩，制造"第一"噱头

**合规标准**：
- 字号不小于主文字1/2
- 位置与主信息同区域
- 颜色与背景有明显对比
- 视频展示时间充足

详细标准参考：`references/big-small-text.md`

### Step 5: 数据宣称证据链审查
涉及排名、第一、销量、用户量等，审查：
- 数据来源是否权威？
- 统计口径/周期是否明确？
- 第三方证明是否具备？

### Step 6: 视觉元素审查
- 地图：是否使用标准地图（审图号）
- 医生/专家形象：是否真实身份+资质
- 儿童：不得变相代言

### Step 7: 行业专项审查
根据Step 2确定的行业，加载对应reference：

| 行业 | Reference文件 |
|------|---------------|
| 儿童/母婴产品 | references/children.md |
| 教育培训 | references/education.md |
| 药品/医疗器械 | references/medical.md |

典型案例参考：`references/cases.md`

### Step 8: 风险综合评级与输出

**快审输出格式**：
```
## 快审结果

**广告定性**：是否具有商业广告属性 / 是否属于重点行业
**明显高风险点**：
1. [风险描述] → [法律依据] → [建议]

**暂定结论**：🔴不建议发布 / 🟡修改后再审

**修改建议**：原文...建议...
```

## 重点审查清单

### 儿童/母婴产品
- 是否使用不满10周岁儿童代言？
- 是否有不当发育/教育功效宣称？
- 是否标注适用年龄和安全警示？

### 教育培训
- 是否涉及"保过"、"保分"等效果保证？
- 是否涉及升学率/排名宣称？
- 是否利用受益人名义推荐？

### 医疗器械/药品
- 是否说明治愈率/有效率？
- 是否使用医生/患者名义证明？
- 是否标注广告批准文号？

### "大字小字"（2026年重点整治）
- 是否存在大小字号明显差异？
- 限制条件是否显著标明？
- 是否弱化对消费者不利信息？
