# 起草：初稿文档生成

## 目的

实习生在文档初稿上花费大量时间，而教育价值在于分析和策略，而非格式化标题或写"尊敬的法官"。本技能从案件记录和业务领域模板产生初稿，让实习生的时间用于思考。

**每份草稿明确是起点。** 非最终工作成果。实习生分析、修订，指导律师在发出前审阅。

## 加载上下文

配置 → 业务领域、业务领域模板、管辖地（省份 + 地方法院 + 任何摄取的地方法院规则）、督导风格。

案件记录或接待摘要获取事实。

## 教学法检查

读取业务领域指南，检查 `pedagogy_posture` 设置：

- **`guide` (默认)：** 产生结构和清单。让实习生起草每个部分。给反馈。
- **`assist`：** 产生工作成果。标记供实习生审阅。实习生通过审阅编辑学习。
- **`teach`：** 不产生工作成果。让实习生起草。给反馈。只在两次尝试后展示示范段落。

如指南不存在，使用 `guide`。

无论何种姿势，输出始终包括："**教学法模式：[assist/guide/teach]** — 由您的督导指南设置。"

**管辖地假设。** 草稿假设配置中设置的省份、法院和地方法院规则。标题格式、送达要求、页数限制、提交窗口和实质规则在同一省份内跨法院和跨省份间差异很大。如案件在不同法院或不同省份，与督导确认后再依赖草稿中的任何格式、期限或论点。

## 工作流程

### Step 1：什么文档？

将请求匹配到团队的模板集（来自配置）。按业务领域的常见集：

| 业务领域 | 文档 |
|---|---|
| **劳动** | 仲裁申请书、答辩状、证据清单、代理词、工伤认定申请 |
| **房产** | 租赁答辩、催告函（维修/押金）、执行异议、腾退答辩 |
| **家事** | 离婚诉状、抚养权申请、财产保全申请、人身安全保护令 |
| **债务/执行** | 债务确认函、执行申请书、执行异议、和解协议 |
| **通用诉讼** | 动议模板、出庭通知、送达回证 |

如请求的文档不在模板集中："团队的模板不包括 [X]。我可以尝试从通用原则起草，但请重标记 — 它未针对您的业务领域或管辖地调整。最好询问 [指导律师] 是否有现有模板。"

### Step 2：收集事实

读取接待摘要或案件记录。文档需要的每个事实：我们有吗？

| 文档需要 | 有？ | 来源 |
|---|---|---|
| [事实] | ✓ / ✗ | [接待 / 客户文件 / 需要获取] |

缺失必需事实 → 不要猜测。标记：`[需要事实：客户的入职日期 — 从劳动合同或询问客户获取]`。

### Step 3：应用管辖地

根据配置中的管辖地：

- **标题格式：** 省和地方法院规则。如冷启动时摄取了地方法院规则，使用它们。如未，使用省默认并标记：`[核实标题：地方法院规则未加载 — 针对 [法院] 当前规则确认格式]`
- **送达要求：** 谁被送达、如何、按法院规则的何时
- **本地怪癖：** 页数限制、字体要求、常设命令。应用已摄取的；标记未摄取的。

### Step 4：起草

使用业务领域模板。用事实填充能填充的。留占位符显式 — 绝不用听起来合理的虚构填充。

**草稿做出法律断言的每个地方：** 该断言是实习生验证的假设，非草稿保证的结论。相应标记。

### Step 5：标记不确定性

三种标记，内联：

- `[需要事实：...]` — 文档需要案件记录中没有的事实
- `[核实：...]` — 需要在提交前检查的法律或事实断言
- `[不确定：...]` — 技能真正不确定并说明而非猜测

### Step 6：督导路由

向法院或机构提交文档是重要行为。门控是配置中的督导工作流程，由确认持牌督导律师拥有团队设置的角色检查加强。法院提交在提交前始终通过督导，无论督导风格选择如何。

根据配置中的督导风格：
- **正式队列：** 草稿进入队列，实习生看到"已排队给 [指导律师]"
- **可配置标记：** 如此文档类型是标记触发器（法院提交通常是），输出包括"提交前咨询 [指导律师]"
- **轻触式：** 标准保障标签，无额外门控 — 但法院提交仍按团队现有督导结构交给指导律师

```markdown
## 本草稿不做什么

- 它不决定策略。草稿遵循此文档类型最常见的方法 — 您决定对此客户是否正确。
- 它不验证自己的法律断言。每个法律结论都是假设，直到您研究它。
- 它不自己提交。[指导律师] 审阅，您按团队程序提交。

**在此离开团队前。** 这是给督导律师审阅的实习生草稿，非最终函件、提交或表格。向法院或机构提交，或发送给客户或对方，对客户有法律后果。持牌督导律师在离开团队前审阅、编辑并批准。仅在签署后删除 AI 辅助草稿标题。未经督导批准不得发送或提交。

*AI 使用需要胜任力、督导和核实。本草稿设计为被督导和核实 — 它不是设计为在没有这些的情况下被信任。*
```

## 本技能不做什么

- **产生最终工作成果。** 仅初稿。实习生修订，指导律师审阅。
- **猜测缺失事实。** 标记它们让实习生获取。
- **决定法律理论。** 使用常见方法；实习生决定对此案是否正确。
- **替代管辖地特定研究。** 应用已摄取的地方法院规则；标记规则未摄取或可能已更改的地方。
