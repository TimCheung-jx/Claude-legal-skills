# Ramp：周期入职

## 目的

每个周期，团队失去整个工作力并从头重建。新实习生需要学习程序、案件管理、提交惯例和业务领域基础，才能发挥作用。传统上这需要数周阅读 PDF 和问指导律师每个周期同样的问题。

本技能是引导式演练。它读取指导律师在冷启动时上传的内容 — 手册、提交指南、地方法院规则 — 并互动教授，带实践练习让实习生在真实客户上线前在低风险环境中试用工具。

**受众：实习生。** 指导律师不运行此技能（他们运行 `/cold-start-interview`）。

## 加载上下文

配置 → 团队档案、业务领域、管辖地、手册路径、督导风格、业务领域模板。

如该文件缺失或仍有占位符："团队尚未设置。请 [督导指导律师] 先运行 `/cold-start-interview`。"

## 演练

### 开场

> 欢迎来到 [团队名称]。我将带您了解此团队的工作方式以及如何使用这些工具 — 大约二十分钟，您可以随时暂停。结束时您将运行过实践接待、起草过实践文档，并且知道接到第一个真实案件时该做什么。
>
> 一件事先说明：我生成的所有内容都是起点，非最终答案。您做分析。[指导律师] 按督导风格审阅您的工作。我处理格式化和初稿，让您把时间花在 lawyering 上，而非第二十遍写"尊敬的法官"。

### Part 1：此团队（5 分钟）

从配置和摄取的手册读取。互动涵盖：

- **业务领域** — 团队处理什么、不处理什么（以及有人带着超出范围的问题走进来时转介到哪里）
- **客户** — 他们是谁、他们面临什么、语言
- **管辖地** — 哪些法院、哪些法官、当地怪癖是什么
- **案件管理** — 案件如何跟踪、文件在哪里、记录良好的案件是什么样子
- **督导** — 此团队中审阅如何工作（按配置中的督导风格）。具体："在任何东西发给客户或法院前，[它进入审核队列 / 您咨询 X 指导律师 / 等]"

不要 lecturing — 检查理解。"所以如果客户带着腾退通知进来，但也提到他们没有劳动合同，您怎么做？"（答案：两个问题都在接待中记录；劳动法问题可能需要转介或标记给指导律师，取决于团队范围。）

### Part 2：命令（5 分钟）

演练实习生将实际使用的每个命令：

| 命令 | 何时使用 | 您得到什么 |
|---|---|---|
| `/client-intake` | 客户访谈 | 格式化的案件摘要，问题发现，利益冲突标记，分流 |
| `/draft [doc type]` | 需要常见文档的初稿 | 业务领域模板，从案件记录填充 — *起点，非最终* |
| `/memo` | 需要内部分析案件 | IRAC 格式备忘录，研究缺口标记 |
| `/research-start [issue]` | 开始法律研究 | 路线图：要检查的法规、案例法领域、搜索词 — *线索，非权威引用* |
| `/status [audience]` | 向某人更新案件 | 针对客户 / 指导律师 / 法院的定制摘要 |
| `/client-letter [type]` | 常规函件 | 预约确认、文件请求、状态更新，来自模板 |

每个：它做什么、它明确不做什么、实习生在依赖前核实什么。

### Part 3：实践练习（8-10 分钟）

**低风险。模拟客户。真实工具。**

**练习 1 — 实践接待：**
> 这是一个模拟客户场景：[业务领域适当的假设 — 例如，对于房产团队，"Maria 上周二收到了 3 天搬离通知。她失业后拖欠了两个月租金。公寓自 11 月以来暖气就坏了。她有两个孩子。"]
>
> 运行 `/client-intake` 并像我是 Maria 一样访谈我。我会像 Maria 那样回答。结束时，查看它产生的案件摘要 — 它发现了哪些问题？它抓到居住条件抗辩了吗？

复盘：接待抓到了什么、*实习生* 应该更深入探查什么、什么被标记给指导律师。

**练习 2 — 实践起草：**
> 使用 Maria 的接待，运行 `/draft eviction-answer`。您会得到初稿。
>
> 阅读它。什么是对的？什么是错的？在展示给 [指导律师] 前您会改什么？

要点：草稿 competent 但非最终。实习生学会批判性阅读，而非接受。

**练习 3 — 研究路线图：**
> 运行 `/research-start "[省份] 腾退中的居住条件抗辩"。您会得到路线图 — 法规、案例法领域、搜索词。
>
> 这些引用都未验证。这是故意的。从路线图中挑选一个法规并告诉我您会如何验证它是现行且适用的。

要点：`/research-start` 是起点，非引用。实习生仍做研究。

### Part 4：核实习惯（2 分钟）

重要的习惯：

- **每个输出都是起点。** 如它未经您批判性阅读就发给客户或法院，出了问题。
- **在放入任何东西前核实每个引用。** `/research-start` 给线索，非权威。
- **检查管辖地特定细节。** 插件从设置知道您的省份，但地方法院怪癖会变化 — 针对当前地方法院规则双重检查。
- **当不确定时，它会说。** 如输出有 `[不确定：...]` 标记，那是研究或问指导律师的提示，非删除标记继续。
- **[配置中的督导提醒]** — 什么在发出前被审阅，以及如何。

### 结束

> 就这些。您运行过接待、起草过文档、构建过研究路线图。您的第一个真实案件会感觉类似，只是客户是真实的，指导律师在阅读您的工作。
>
> 单页参考卡：`/ramp --card`

## `/ramp --card`

按单页卡规范生成单页实习生参考卡。内容：

- 命令（Part 2 的表格，精简）
- Claude 能帮什么 / 不能帮什么（起点可以，最终工作成果不行，权威引用不行）
- 核实习惯（Part 4 的要点）
- 卡住时问谁（配置中的指导律师姓名）

可打印。一页。第一天分发。

## 本技能不做什么

- 替代指导律师的方向。它覆盖程序和工具；指导律师覆盖判断、策略和您只能通过观察优秀的人做事才能学到的东西。
- 教授实体法。业务领域 *方向*，非教义的课程。
- 认证实习生已准备好。指导律师决定实习生何时接手真实案件。
